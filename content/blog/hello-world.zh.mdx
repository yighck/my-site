欢迎来到我的博客！这是我的第一篇文章。我创建这个站点是为了分享技术笔记、项目经验以及日常学习所得。

## 为什么要写？

写作是巩固知识理解的最佳方式之一。当你用自己的话解释一个主题时，你会很快发现知识中的盲区。

> "如果你不能简单地解释它，说明你理解得还不够好。" — 阿尔伯特·爱因斯坦

## 内容规划

这个博客将涵盖：

- **Web 开发** — React、Next.js、TypeScript 以及现代开发工具
- **技术笔记** — 来自实际项目的技巧、经验与教训
- **文档** — 日常使用工具的指南和参考

## 技术栈

本站使用以下技术构建：

- [Next.js 16](https://nextjs.org) — 使用 App Router 的 React 框架
- [Tailwind CSS v4](https://tailwindcss.com) — 工具类优先的 CSS 框架
- MDX — 支持 JSX 的 Markdown
- Shiki — 代码块语法高亮

一个简单的示例：

```tsx
export default function HelloWorld() {
  return (
    <div>
      <h1>你好，世界！</h1>
      <p>欢迎来到我的博客。</p>
    </div>
  );
}
```
