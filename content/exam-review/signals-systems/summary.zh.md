# 信号与系统 知识点总结

## 第一章 信号与系统基本概念

### 1.1 信号分类
- **连续信号** vs **离散信号**
- **周期信号**：x(t) = x(t+T)，最小 T 为基波周期
- **偶信号**：x(t)=x(-t)；**奇信号**：x(t)=-x(-t)

### 1.2 基本信号
- **单位冲激 δ(t)**：∫δ(t)dt=1, δ(t)=0 (t≠0)
- **单位阶跃 u(t)**：u(t)=1 (t>0), u(t)=0 (t<0)
- **关系**：u(t) = ∫δ(τ)dτ, δ(t) = du(t)/dt

## 第二章 连续系统的时域分析

### 2.1 LTI 系统
- **线性**：叠加性 + 齐次性
- **时不变性**：输入延时 → 输出等量延时
- **卷积**：y(t) = x(t) * h(t) = ∫x(τ)h(t-τ)dτ

### 2.2 卷积性质
- 交换律、结合律、分配律
- x(t)*δ(t) = x(t)
- x(t)*δ(t-t₀) = x(t-t₀)

## 第三章 傅里叶变换

### 3.1 连续时间傅里叶变换 (CTFT)
- **正变换**：X(jω) = ∫x(t)e^(-jωt) dt
- **逆变换**：x(t) = (1/2π) ∫X(jω)e^(jωt) dω

### 3.2 常见变换对
| 信号 x(t) | 变换 X(jω) |
|-----------|------------|
| δ(t) | 1 |
| 1 | 2πδ(ω) |
| e^(jω₀t) | 2πδ(ω-ω₀) |
| u(t) | πδ(ω) + 1/(jω) |
| e^(-at)u(t), a>0 | 1/(a+jω) |

### 3.3 傅里叶变换性质
- **时移**：x(t-t₀) ↔ e^(-jωt₀)X(jω)
- **频移**：e^(jω₀t)x(t) ↔ X(j(ω-ω₀))
- **卷积定理**：x₁*x₂ ↔ X₁·X₂
- **对偶性**

## 第四章 拉普拉斯变换

### 4.1 定义
- X(s) = ∫x(t)e^(-st) dt, s = σ + jω
- 收敛域 (ROC)：使积分收敛的 s 值范围

### 4.2 常见变换对
| x(t) | X(s) | ROC |
|------|------|-----|
| δ(t) | 1 | 全 s 平面 |
| u(t) | 1/s | Re(s)>0 |
| e^(-at)u(t) | 1/(s+a) | Re(s)>-a |
| tⁿu(t) | n!/s^(n+1) | Re(s)>0 |

### 4.3 系统函数 H(s)
- H(s) = Y(s)/X(s)
- **极点**决定系统稳定性
- 所有极点在左半平面 → 稳定

## 第五章 Z 变换（离散系统）

### 5.1 定义
- X(z) = Σ x[n]z^(-n)
- 是离散时间傅里叶变换的推广

### 5.2 常见变换对
| x[n] | X(z) |
|------|------|
| δ[n] | 1 |
| u[n] | z/(z-1) |
| aⁿu[n] | z/(z-a) |

### 5.3 系统函数 H(z)
- H(z) = Y(z)/X(z)
- 因果稳定系统：所有极点在单位圆内
