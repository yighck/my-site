欢迎！这个站点是我的个人博客和知识库。这里包含技术笔记、项目文档和各种指南。

## 这是什么站点？

这是一个**个人知识库**，使用以下技术构建：

- **Next.js 16** — 使用 App Router 的 React 框架
- **MDX** — 支持 JSX 的 Markdown，用于富文本内容
- **Tailwind CSS** — 工具类优先的 CSS 框架
- **Shiki** — 精美的代码语法高亮

## 站点结构

内容分为两个主要板块：

### 博客

技术文章、项目笔记和学习总结。文章按日期排序，支持标签筛选。

### 文档

按主题组织的结构化参考资料。文档页面配有侧边栏导航，方便浏览。

## 如何浏览

- 使用**顶部导航栏**在博客和文档之间切换
- 使用**搜索**（点击搜索图标或按 `Cmd/Ctrl + K`）快速查找内容
- 在文档页面使用**侧边栏**浏览主题
- 点击右下角按钮切换**暗色模式**
- 使用导航栏中的按钮切换**语言**

## 添加内容

所有内容使用 MDX 格式编写。详见 [撰写博客文章](/docs/guides/writing-blog) 和 [添加文档](/docs/guides/adding-docs) 指南。
