Welcome! This site serves as my personal blog and knowledge base. Here you'll find technical notes, project documentation, and guides.

## What Is This Site?

This is a **personal knowledge base** built with:

- **Next.js 16** — React framework with App Router
- **MDX** — Markdown with JSX support for rich content
- **Tailwind CSS** — Utility-first styling
- **Shiki** — Beautiful code syntax highlighting

## Site Structure

The content is organized into two main sections:

### Blog

Technical articles, project notes, and learning summaries. Posts are sorted by date and tagged for easy filtering.

### Documentation

Structured reference material organized by topic. Documentation pages have a sidebar navigation for easy browsing.

## How to Navigate

- Use the **top navigation** to switch between Blog and Docs
- Use **search** (click the search icon or press `Cmd/Ctrl + K`) to find content quickly
- Use the **sidebar** on documentation pages to browse topics
- Toggle **dark mode** with the button in the bottom-right corner
- Switch **language** using the button in the navigation bar

## Adding Content

All content is written in MDX format. See the [Writing Blog Posts](/docs/guides/writing-blog) and [Adding Documentation](/docs/guides/adding-docs) guides for details.
